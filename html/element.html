초기화방법입니다.
Ex) Elite Active 65t 기준
1. 우측 이어버드 전원을 킵니다.
(적색 불빛은 전원이 꺼짐을 의미합니다. 적색 불빛을 확인 후, 전원을 다시 켜주시기 바랍니다.
2. 이어버드 LED가 녹색 -> 파란색 -> 보라색으로 나올 때 까지 버튼을 계속 누릅니다. (약 10초)
3. 버튼에서 손을 떼고 파란 불빛이 나오면 스마트폰에 블루투스를 활성화하여 연결해주세요.
4. 좌측 이어버드는 좌측 볼륨버튼으로 다시 켜주시길 바랍니다.

세부 사항은 아래 링크로 들어가신 후,
사용하고 계시는 제품과 스마트폰 선택하시면 페어링 방법 상세히 나옵니다.
참고 부탁드립니다.
https://www.youtube.com/watch?v=iAVhMg6h6a8&t=45s
감사합니다.
Ex) Elite Active 65t 기준
좌측 이어버드 단독 사용은 불가 합니다.
우측 이어버드 단독 사용가능하나 충전단자에서 꺼낼시 우측이어버드만 꺼내서 사용부탁드립니다.
2017년 현재 출시되는 모든 WD 외장하드는 GPT 파티션으로 포멧되어 있습니다.
스마트 TV 의 경우 GPT 를 대부분 지원하지 않습니다. (일부 최신 TV 호환 가능)
TV 에서 사용하시려면, windows 7 이상인 컴퓨터에서 인식되는지 확인 하신 후
파티션을 MBR 로 변경하시면 됩니다.
변경방법은 대원CTS 홈페이지 [ 기술지원 -> 통합 자료실 -> MBR 포멧방법 ] 참고바랍니다.
단, 이 작업을 하시면 외장하드의 모든 data는 삭제되는 점 유의 바랍니다.
MBR 변경하면 2TB 까지만 인식이 됩니다. 그 이상의 용량은 사용할 수 없는점 유의바랍니다.
TV는 외장하드 사용시 안전제거가 되지 않으므로 제거시 유의 바랍니다.
외장하드 고장의 원인이 될 수 있습니다.
